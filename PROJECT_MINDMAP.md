## 9. 通信与消息流

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         通信机制详解                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Agent间通信方式:                                                          │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │  1. 直接调用 (Direct Call)                                         │  │
│   │     AgentA ──────▶ AgentB.execute_tool()                           │  │
│   │                                                                     │  │
│   │  2. 消息队列 (Message Queue)                                       │  │
│   │     AgentA ──▶ Queue ──▶ AgentB                                     │  │
│   │                                                                     │  │
│   │  3. 事件驱动 (Event Driven)                                        │  │
│   │     AgentA ──▶ Event ──▶ [AgentB, AgentC, ...]                      │  │
│   │                                                                     │  │
│   │  4. 共享上下文 (Shared Context)                                    │  │
│   │     ┌─────────────┐                                                │  │
│   │     │ Unified     │◀── AgentA                                       │  │
│   │     │ Context     │──▶ AgentB                                       │  │
│   │     └─────────────┘    └──▶ AgentC                                 │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│   消息格式:                                                                 │
```
