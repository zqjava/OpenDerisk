# OpenDerisk Bun Package

Fast, TypeScript-native package for OpenDeRisk AI-Native Risk Intelligence Systems.

## Installation

### Global installation
```bash
bun install -g openderisk
```

### Local installation
```bash
bun add openderisk
```

### Run without installation
```bash
bunx openderisk
```

## Usage

```bash
# Show help
openderisk --help

# Start OpenDerisk
openderisk

# Start server
openderisk-server

# Update to latest version
openderisk --update
```

## Features

- 🚀 **Fast**: Built with Bun runtime for optimal performance
- 📦 **TypeScript**: Full TypeScript support
- 🔧 **Auto-setup**: Automatic uv and dependency installation
- 🔄 **Auto-update**: Built-in update mechanism

## Requirements

- Bun >= 1.0.0
- Python >= 3.10 (installed automatically)
- Git

## Why Bun?

This package uses Bun for:
- Faster startup times
- Native TypeScript support
- Better cross-platform compatibility
- Smaller bundle size
